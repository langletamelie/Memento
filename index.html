Gestion de documents

Commandes Résultats Commandes Résultats Commandes Résultats
man ls Affiche le manuel ls Affiche le contenu du dossier courant ls -a Affiche le dossier et fichiers cachés
ls -l Version longue du dossier ls -al Version longue avec fichiers cachés pwd Permet de se situer(chemin d'accès)
cd "nom du dossier" Permet de se déplacer dans le dossier cd .. Retourner au dossier parent cd ~ Retourner à la racine
cd ../.. Permet de retourner au dossier gd-parent mkdir website ou mkdir Documents/website Création du dossier website dans Documents mkdir -p Memento/assets (dans website) Permet de créer le dossier parents(Memento) en même tps que le sous-dossier
mkdir css img js création 3 dossiers en même temps touch index.html créer fichier index.html atom index.html ouvre le fichier index.html sur atom

Git et Github

Pour sauvegarder les fichiers que l'on créer, on peut utiliser la solution Github. On se positionne dans le dossier et l'on fait plusieurs manipulations:

  • git init
  • touch NomDuFichier.html NomDuFichier.css
  • git add NomDuFichier.html NomDuFichier.css
  • atom NomDuFichier.html
  • git commit -m "commentaire dur la modification apportée"
  • git status
  • git log
  • git remote add origin url du repo sur github
  • git push -u origin master
